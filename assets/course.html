課程介紹

　　我們的課程眾多且豐富：有一周一堂的社課，以及每周一到五都有的放課供你參加、學習！

社課

　　社課是在每個禮拜五下午的第一堂課，所有社員皆需要來參加，我們的上課地點在資源大樓（夢紅樓旁）三樓的電腦教室二，會有專業的講師在社課中教授一些基礎的程式相關知識，即使之前鮮少接觸程式，也可以在社課中快速掌握相關概念！我們這學期會在社課中教學的內容主要有Python基礎語法、基礎應用，以及最後留幾堂課的時間讓社員們進行成果發表，藉這個機會互相交流、切磋！

放課

　　放課可以說是資訊社的精華部分，我們會在每個禮拜的一到五放學後的晚上6:00 ~ 8:00跟成功、中山、景美在資源大樓三樓電腦教室二舉行，這邊的教學內容會較社課來的更為進階，多了更多實作的部分，不管你是想要來精進自己的程式技術，又或是想要來跟講師討論、切磋，甚至是想要來把講師電爛，放課都是你的不二選擇！

本課程將介紹 C# 語言的基礎語法與程式設計方法，內容包含變數、流程控制、陣列、方法及物件導向的基本概念。課程重點在於建立程式設計思維，並透過語法與結構的練習，掌握撰寫程式的核心能力
DC Bot 顧名思義，是社群軟體 Discord 的機器人。以平常在 Discord 常常會看到的「YEE 式機器龍」為例，可以讓我們在使用 DC 時除了聊天，能夠有更多豐富的擴充功能，例如：能夠選擇職業，和好友一同合作的 RPG 角色扮演遊戲、更方便的管理伺服器，又或是在語音頻道撥放音樂。

在這堂放課，會從 JavaScript 的基礎語法帶入，認識這個跨平台、用途廣泛，也適合入門的程式語言，並從零開始帶大家製作屬於自己的 Discord 機器人。無論是初學者，或是想要擴展自己能力的電神，都歡迎來一起學習！
想設計出好看的網頁嗎？想寫出有趣的網頁小遊戲嗎？那麼你應該來這門課程！本課程介紹網頁前端基礎技術，透過實作練習，將學會製作美觀又互動的網頁，例如個人簡介頁、作品集網站等，為進一步學習前端開發打下基礎。

課程內容包含：
  • 網頁結構介紹
  • HTML
  • CSS
  • JS
本課程將讓你學會網路的組成、資料傳輸方式、網路安全以及日常生活中網路技術的應用。課程重點在於理解「網路是什麼」以及「它如何影響我們的生活」。

學習目標：
  • 理解網路的基本概念與歷史發展。
  • 認識常見網路設備與通訊協定。
  • 掌握資料在網路中的傳輸原理。
  • 能夠分析簡單的日常網路現象，並應用基礎網路知識。
